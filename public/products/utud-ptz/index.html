УТУД-ПТЗ

Универсальное устройство обработки данных — высокопроизводительное решение для различных применений в транспорте, сельхозтехнике и специальной технике.

Основные возможности

  • Высокопроизводительный процессор iMX8M Plus ARM Cortex-A53 с частотой 1.6 ГГц
  • Поддержка GPS/ГЛОНАСС L1/L2 с технологией RTK для высокоточного позиционирования
  • Множественные интерфейсы: USB 3.0, USB 2.0, Ethernet, 3xCAN, WiFi, Bluetooth 5.2
  • Видеовыходы с разрешением до 1920x1080
  • Промышленная защита IP65
