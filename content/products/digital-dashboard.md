# Цифровая комбинация приборов

**Цифровая приборная панель** — современное решение для отображения информации в автомобилях, спецтехнике и промышленном оборудовании.

## Основные возможности

- **10.1" дисплей** с высоким разрешением 1920×720
- Мощный процессор **iMX8M Plus** для плавной работы интерфейса
- **4GB LPDDR4 RAM** и **16GB eMMC** для быстрой работы
- **CAN-интерфейс** для интеграции с бортовыми системами

## Технические характеристики

| Параметр | Значение |
|----------|----------|
| **Дисплей** | 10.1" 1920×720 |
| **Процессор** | iMX8M Plus |
| **Память** | 4GB LPDDR4, 16GB eMMC |
| **Интерфейс** | CAN-интерфейс |
| **Рабочая температура** | -40...+65°C |

## Области применения

- **Автомобили** — замена традиционных аналоговых приборов
- **Спецтехника** — отображение рабочих параметров техники
- **Промышленное оборудование** — мониторинг состояния оборудования

## Преимущества

- ✅ **Высокое разрешение** — четкое отображение информации
- ✅ **Мощная платформа** — быстрая обработка данных
- ✅ **Настраиваемый интерфейс** — адаптация под конкретные задачи
- ✅ **Промышленная надежность** — работа в широком температурном диапазоне

## Функциональность

- Отображение скорости, оборотов, температуры
- Индикация предупреждений и ошибок
- Настраиваемые виджеты и экраны
- Поддержка различных протоколов CAN
